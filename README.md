## Installation
Follow the steps below

#### - On the Frontend:
> 1. `pnpm install && pnpm dev`
#### - On the Backend:
> 1. `pnpm install`
> 2. `pnpm db:generate`
> 3. `pnpm db:migrate`
> 4. `pnpm db:seed`
> 5. `pnpm dev`
